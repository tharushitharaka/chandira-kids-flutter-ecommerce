# Chandira Kids Flutter App

A modern mini e-commerce Flutter application developed for the CyphLab Flutter Developer Intern Practical Task.

## Features

- Splash Screen
- Product Listing Screen
- Product Details Screen
- Shopping Cart Management
- Profile Screen
- Product Search
- Category Filtering
- Dark Mode Support
- Responsive Mobile UI

## Tech Stack

- Flutter
- Provider (State Management)
- Shared Preferences (Local Storage)

## Project Structure

```
lib/
├── data/
├── models/
├── providers/
├── screens/
├── widgets/
├── utils/
└── main.dart
```

## Getting Started

### Clone the repository
```bash
git clone https://github.com/tharushitharaka/chandira-kids-flutter-ecommerce.git
```

### Install dependencies
```bash
flutter pub get
```

### Run the application
```bash
flutter run
```
